Students involved in the 'Solar Siblings' project learn about and conduct authentic Astronomy. The first things they learn about include:

  • How to locate objects in the sky using ‘Right Ascension’ (RA) and ‘Declination’ (Dec)
  • How to process the 3 images requested to make it a full colour image (as shown in the image to the right)

Once students learn how to do this they will then be given star clusters, some never before analysed, to determine their distances and contribute to real scientific knowledge.

 To do this students need to learn how to request an image from the LCOGT telescopes located around the world in locations that include: Hawaii and Texas, USA; Coonabarabran, Australia; Cerro Tololo, Chile.
